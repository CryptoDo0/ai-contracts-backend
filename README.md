# Base Project

### Start Backend

Make directory for logs 
```bash
sudo mkdir /var/log/baseapp/
sudo chown -R name:name /var/log/baseapp/
```

Make directory for images 
```bash
sudo mkdir /var/log/baseapp/
sudo chown -R name:name /var/log/baseapp/
```

Go to backend module
```bash
cd backend/
```
Install package
```bash
npm install
```

Run backend dev
```bash
npm run start:dev
```

Go to flatten module and install
```bash
cd backend/flatten
npm install
```
